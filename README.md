PDF Annotation Tool
## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Language**: JavaScript (no TypeScript)
- **Font**: [Geist](https://vercel.com/font) using `next/font`
